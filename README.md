## What I'm Working On

- Designing agent swarm workflows for coordinated research, analysis, and decision-making across real-world workflows
- Creating LLM-powered Chrome extensions that perform actions on the web and help users curate custom, task-specific workflows
- Exploring memory and context management strategies
- Exploring agent-to-agent architectures (A2A, ACP), including discovery, coordination, and communication protocols between agents
- Building tooling and abstractions that make agentic engineering faster, safer, and more accessible for builders
- Designing and deploying agentic systems on cloud infrastructure (AWS, GCP)
- …and continuously experimenting with new ideas, prototypes, and system designs across the agentic stack
